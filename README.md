🚗 Car Type Classifier (AI)
📌 Description

This project is a machine learning model that classifies car types (SUV, sedan, hatchback, pickup, other) from images.

🤖 Features
Image-based car classification
Supports multiple car categories
Real-time prediction from input images

🧠 Model Overview
Uses a trained model for image classification
Takes an input image and predicts the car type
Displays prediction result directly on the image

📂 Dataset
The model was trained using a dataset of labeled car images including categories such as SUV, sedan, hatchback, pickup, and others.
Due to size limitations, the full dataset is not included in this repository.

🛠 Technologies Used
Python
Machine Learning
PyTorch
Matplotlib
Jupyter Notebook
